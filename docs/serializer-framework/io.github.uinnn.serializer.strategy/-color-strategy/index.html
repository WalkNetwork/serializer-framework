Default strategy of all serializations formats! When the Kotlin Serialization encode a Serializable object this will try to encode all strings or list of strings by replacing then to '&', and decoding to 'ยง'

This object is equals to:

override fun encodeString(descriptor: SerialDescriptor, index: Int, value: String) =
  value.replace('ยง', '&')

override fun decodeString(descriptor: SerialDescriptor, index: Int, value: String) =
  value.replace('&', 'ยง')
