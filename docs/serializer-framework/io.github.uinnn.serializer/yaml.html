fun <T : Any> yaml(file: File, model: KClass<T>, format: AlterableStringFormat = DefaultYamlStrategyFormat): YamlFile<T>
Content copied to clipboard

Constructs and loads a YAML file. The default format for YAML files is DefaultYamlStrategyFormat, thats contains a set of serializers and ColorStrategy as a backend strategy, thats replaces all '§' to '&' and vice-versa in strings and lists of strings!

Note:

This need the model kclass with all default constructors or will be throw a error, because this just create a instance using Kotlin Reflect.

Example:

@Serializable
data class Settings(var name: String) // will be throw a error.
@Serializable
data class Settings(var name: String = "uinnn") // works!
